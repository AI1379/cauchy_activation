# 基于 Cauchy 核函数的轻量化残差混合机制探索

## 一、项目主要内容简介

本项目的研究对象为 **Cauchy 残差混合器**（Cauchy Residual Mixer）——一种基于 Cauchy 核函数的深层神经网络残差融合机制。该研究属于深度学习优化理论与数值逼近的交叉领域。

### 1.1 核心数学问题

标准残差网络的递推关系为：

$$h_{t} = h_{t-1} + F_t(h_{t-1})$$

本项目研究的核心问题是：**如何通过引入基于层间距离的加权机制，改进深层网络中的信息流动？**

具体地，我们提出如下修正的递推关系：

$$h_t = F_t(h_{t-1}) + \sum_{i=0}^{t-1} w_{t,i}(\lambda, d) \cdot h_i$$

其中权重函数 $w_{t,i}$ 采用 Cauchy 核形式：

$$w_{t,i} = \frac{\lambda / ((t-i)^2 + d^2)}{\sum_{j=0}^{t-1} \lambda / ((t-j)^2 + d^2)}$$

### 1.2 研究内容

本项目主要包含以下理论探索与实验验证：

1. **Cauchy 核函数在残差加权中的应用**：研究 Cauchy 分布的长尾性质对深层网络梯度传播的影响，探索其相比高斯核、指数核的理论优势

2. **函数逼近能力研究**：在多元复杂函数上分析不同残差结构的逼近误差，研究 Cauchy 残差混合对神经网络万能逼近性质的影响

3. **偏微分方程数值求解**：在 PINN（Physics-Informed Neural Networks）框架下，探索该机制对椭圆型 PDE 求解精度和收敛性的影响

4. **分类问题的泛化性能**：在图像分类任务中研究该机制对模型泛化能力的影响

### 1.3 Cauchy 核的数学理论基础

#### 1.3.1 从复分析视角理解 Cauchy 核

Cauchy 核源于复分析中的 **Cauchy 积分公式**（Cauchy Integral Formula）。对于复平面上的解析函数 $f(z)$，有：

$$f(z) = \frac{1}{2\pi i} \oint_{\Gamma} \frac{f(\zeta)}{\zeta - z} \, d\zeta$$

其中 $\frac{1}{\zeta - z}$ 称为 **Cauchy 核**。该公式表明，解析函数在任一点的值可由其边界值的加权积分完全确定——权重由 Cauchy 核给出。

将 Cauchy 核限制在实轴上，考虑 $\zeta = x \in \mathbb{R}$，$z = x_0 + i\gamma$（复数点），则：

$$\frac{1}{\zeta - z} = \frac{1}{x - x_0 - i\gamma} = \frac{x - x_0 + i\gamma}{(x-x_0)^2 + \gamma^2}$$

取虚部得到 **Poisson 核**，取实部得到 **Hilbert 核**。本项目采用的 Cauchy 分布密度函数：

$$p(x) = \frac{1}{\pi} \cdot \frac{\gamma}{(x-x_0)^2 + \gamma^2}$$

正是 Poisson 核在实轴上的表示，它与 Cauchy 核的虚部密切相关。

#### 1.3.2 Cauchy 核的特殊性质

**（1）长尾特性与矩的不存在性**

Cauchy 分布是典型的**重尾分布**（Heavy-tailed Distribution）。其概率密度函数以 $O(1/x^2)$ 的速度衰减，远慢于高斯分布的 $O(e^{-x^2})$。这导致：

- 方差不存在：$\int_{-\infty}^{+\infty} x^2 p(x) \, dx = +\infty$
- 数学期望不存在（除非限定为中心 Cauchy 分布的对称性解释）

在深层网络中，这种长尾特性的潜在优势是：**距离较远的层仍能获得相对较大的权重**，而非快速衰减至零。

**（2）调和函数与均值性质**

Cauchy 核与上半平面的 **Poisson 核** 密切相关：

$$P_y(x) = \frac{1}{\pi} \cdot \frac{y}{x^2 + y^2}$$

Poisson 核是调和函数理论的核心工具，满足**均值性质**（Mean Value Property）：调和函数在一点的值等于其在周围圆周上的平均值。这启发我们思考：**残差混合是否可以视为一种离散化的"加权均值"操作？**

**（3）奇异积分与正则性**

Cauchy 核是**奇异积分核**（Singular Integral Kernel）的原型。在调和分析中，奇异积分算子：

$$Tf(x) = \text{p.v.} \int \frac{f(y)}{x-y} \, dy$$

定义了 **Hilbert 变换**，这是 Fourier 分析中的基本算子。Cauchy 核的奇异性（在 $x = x_0$ 处的一阶极点）使其具有独特的信号处理特性。

**（4）保角变换下的不变性**

Cauchy 核在 **Möbius 变换**（分式线性变换）下具有良好的变换性质，这是复分析中保角映射理论的基础。这种"结构性"可能是 Cauchy 核在多种数学语境中自然出现的原因之一。

#### 1.3.3 本项目选择的数学动机

基于上述分析，本项目选择 Cauchy 核形式（$(t-i)^2 + d^2$ 作为分母）的数学动机包括：

| 数学性质 | 潜在的网络意义 |
|----------|----------------|
| 长尾衰减 $O(1/r^2)$ | 远距离层特征权重衰减较慢 |
| 与调和函数的联系 | 可能与网络的"平滑性"相关 |
| 奇异积分核的原型 | 具有非平凡的信号加权特性 |
| 参数 $d$ 控制奇异性 | 可学习参数调节权重分布形状 |

当然，这些主要是启发性的理论动机，其实际效果需要通过实验验证。

## 二、项目背景

### 2.1 残差网络的发展与局限

残差网络（ResNet）通过跳跃连接（Skip Connection）缓解了深层网络的梯度消失问题，其核心公式为：

$$y = x + F(x)$$

然而，标准残差连接采用简单的等权相加，这种方式在深层网络中可能存在以下局限：

- **信息流动**：深层特征难以有效利用浅层信息
- **权重分配**：所有历史层特征被同等对待，缺乏差异化权重

### 2.2 相关工作与灵感来源

本项目的核心思想受到以下工作的启发：

- **Kimi AttnRes**：Moonshot AI 的 Kimi 模型中提出的注意力残差机制，通过注意力加权融合多层特征
- **DeepSeek mHC**（multi-Head Compression）：DeepSeek 提出的多头压缩技术，用于高效的信息融合

这些方法在大型语言模型中展现了较好的效果，但通常需要较多的计算开销（如注意力机制的 $O(L^2)$ 复杂度）。本项目希望探索一个更基础的问题：**是否可以用更简单的机制实现类似的效果？**

### 2.3 Cauchy 核函数的选择动机

Cauchy 分布（Lorentz 分布）的概率密度函数为：

$$f(x) = \frac{1}{\pi} \cdot \frac{\gamma}{(x-x_0)^2 + \gamma^2}$$

本项目尝试借鉴 Cauchy 函数形式作为距离加权核，用于计算残差混合权重。选择该核函数的考虑因素包括：

- **长尾特性**：相比高斯核，Cauchy 核衰减更慢，理论上可能保留更远的层间信息
- **计算简单**：仅需平方运算，无需指数计算
- **可微分**：便于端到端训练

### 2.4 Cauchy 激活函数（辅助组件）

> 注：Cauchy 激活函数为已有工作，本项目中作为可选的激活函数替代方案进行对比实验，非本项目的核心创新。

Cauchy 激活函数定义为：

$$\phi(x) = \lambda_1 \cdot \frac{x}{x^2 + d^2} + \lambda_2 \cdot \frac{1}{x^2 + d^2}$$

其中 $\lambda_1$、$\lambda_2$、$d$ 均为可训练参数。

## 四、项目研究方案

### 4.1 核心算法设计

#### 4.1.1 Cauchy 残差混合器

对于第 $t$ 层的残差混合，采用 Cauchy 距离加权：

$$h_t = F_t(h_{t-1}) + \sum_{i=0}^{t-1} w_{t,i} \cdot h_i$$

其中权重计算方式为：

$$w_{t,i} = \frac{\lambda / ((t-i)^2 + d^2)}{\sum_{j=0}^{t-1} \lambda / ((t-j)^2 + d^2)}$$

**参数量**：对于 $L$ 层网络，需要 $2(L+1)$ 个额外标量参数（$\lambda$ 和 $d$）。

该设计的基本假设：

- 距离当前层越远的特征，其权重可能应该越小
- 权重衰减的速率和形状可以通过训练学习
- 通过保留对早期层特征的加权连接，可能改善信息流动

### 4.2 实验设计

#### 4.2.1 曲线拟合实验

**目标函数**：
$$z(x,y) = \sin(x)\cos(y) + 0.1\sin(5x)\cos(5y) + \sin(x^2 + y^2)$$

**实验配置**：

| 参数 | 值 |
|------|-----|
| 网络深度 | 30 层 |
| 隐藏层维度 | 128 |
| 训练样本数 | 5000 |
| 训练轮次 | 300 |
| 学习率 | 1e-5 |
| 优化器 | AdamW |

**对比组合**（2×2 因子设计）：

- 残差方式：Standard / Cauchy
- 激活函数：ReLU / Cauchy

#### 4.2.2 PDE 求解实验（PINN）

**目标 PDE**（Poisson 方程）：
$$-\Delta u(x,y) = 2\pi^2\sin(\pi x)\sin(\pi y), \quad (x,y) \in [-1,1]^2$$

**边界条件**：
$$u(x,y) = 0, \quad (x,y) \in \partial[-1,1]^2$$

**解析解**：
$$u^*(x,y) = \sin(\pi x)\sin(\pi y)$$

**损失函数**：
$$L = L_{pde} + w_{bc} \cdot L_{bc}$$

**采样配置**：

| 参数 | 值 |
|------|-----|
| 内部点数 | 4096 |
| 边界点数（每边） | 1024 |
| 训练轮次 | 1200 |
| 学习率 | 1e-3 |

#### 4.2.3 图像分类实验（CIFAR）

**数据集**：CIFAR-10 / CIFAR-100

**网络架构**：自定义 CNN

- 卷积层 + BatchNorm + 激活函数
- 残差块结构
- 全局平均池化 + 分类头

**训练配置**：

| 参数 | 值 |
|------|-----|
| Batch Size | 128 |
| Epochs | 20 |
| Learning Rate | 0.001 |
| Weight Decay | 0.0001 |
| Base Channels | 64 |

### 4.3 评价指标

1. **曲线拟合**：训练/验证 MSE 损失、稠密网格上的 MSE 误差
2. **PDE 求解**：PINN 总损失、PDE 损失、边界损失、与解析解的 MSE
3. **图像分类**：训练/测试准确率、损失曲线
